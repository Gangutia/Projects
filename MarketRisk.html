The purpose of this project is to show how a Stochastic Geometric Brownian Movement Process can be applied to the finance sector, specifically to model Market Risk. As an example we will use the data related to the Standard & Poorâs 500 (S&P 500), the most common, famous and reference index to follow the United States Economy.

We will use this type of process to model the evolution of a random system through time with a non constant random variable, being the growth of this variable exponential and with a non constant volatility as the moment value increases.

This function will allow us to calculate various simulations and trajectories in base of the previous parameters. It is important to mention, that this method is an approximate numerical solution of the Stochastic Differential Equation (SDE).

GBM <- function(S0, mu, sigma, dt, n_sim, n_tra) {
  
  trajectories <- list() # Trajectories storage
  
  # Iterate to generate trajectories
  for (tra in 1:n_tra) {
    
    trajectory <- numeric(n_sim + 1) # Store current trajectory
    
    trajectory[1] <- S0 # Set initial price
    
    
    # Iterate to calculate price changes
    for (t in 1:n_sim) {
      
      # Euler Method
      Z <- rnorm(1, mean = 0, sd = 1) # Standard normal random variable
      
      delta_S <- mu * trajectory[t] * dt + sigma * trajectory[t] * sqrt(dt) * Z # Delta S
      
      trajectory[t + 1] <- trajectory[t] + delta_S # Next price
    }
    
    trajectories[[tra]] <- trajectory # Add current trajectory
  }
  return(trajectories)
}


simulations_trajectories <- GBM(S0, mu, sigma, dt, n_sim, n_tra)

The Value at Risk (VaR) is a measure that will quantify the maximum risk of loss of our active over a given period of time and with a specific confidence level.

VaR_func <- function(alpha) {
  
  var_vec <- numeric(length(simulations_trajectories)) # Storage vector 
  
  # Iterate for each trajectory
  for (tra in 1:length(simulations_trajectories)) {
    trajectory <- simulations_trajectories[[tra]]
    var_vec[tra] <- quantile(trajectory, 1 - alpha)
  }
  
  var_aggregated <- quantile(var_vec, 1 - alpha) # Aggregated VaR
  
  return(var_aggregated)
}


The Conditional Value at Risk (CVaR) is a measure that provides an estimate of the expected value of losses in the worst case scenario, being the weighted average of losses exceeding VaR for the confidence level given.

CVaR_func <- function(alpha) {
  
  cvar_vec <- numeric(length(simulations_trajectories)) # Storage vector
  
  # Iterate for each trajectory
  for (tra in 1:length(simulations_trajectories)) {
    trajectory <- simulations_trajectories[[tra]]
    cvar_vec[tra] <- mean(trajectory[trajectory < VaR_func(alpha)])
  }
  
  es_aggreagted <- mean(cvar_vec) # Aggregated CVaR
  
  return(es_aggreagted)
}


The Maximum Drawdown is the peak-to-trough decline in our active over a given period of time.

max_drawdown <- function() {
  
  max_vec <- numeric(length(simulations_trajectories)) # Storage vector
  
  for (tra in 1:length(simulations_trajectories)) {
    trajectory <- simulations_trajectories[[tra]]
    peak <- cummax(trajectory)
    max_vec[tra] <- max(peak - trajectory)
  }
  
  max_aggregated <- max(max_vec) # Maximum Drawdown

  return(max_aggregated)
}
